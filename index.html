Frequently Asked Questions

here are some of the most frequently asked question of HTML, CSS and bootstrap

Grid: Grid is a two-dimensional layout with rows and column. Grid can help create much more resposive item layout easily. Grid is much more layout focused.

Flexbox: Flex is a one-dimensional layout which any move only in row or only in column at a time. Flex is useful for aligning items. Flex is much more content focused.

Semantic tags gives meaning to the website properties. It is used to make sense of the website both for the developer and the browser. When tags like dev means just a box where a nav means navigation bar or main means the main part of the website ans so on.

Bootstrap:Bootstrap is a open-source CSS framework that is used to create responsive website easily. It comes with many preset designs that can to customized to fit the need of the developer. It is known for it's responsive design and mobile first design.

Tailwind:Tailwind is a relatively new CSS framework that is muh more focused on utilities and known for its customizability. It is relatively small in size compared to bootstrap. Using tailwind one can create very customized website with unique design.

CSS Box model is the how a web browser sees a HTML that s wrapped with CSS. It is a box that contains the HTML component. A CSS box model has 4 parts: Content, padding, border and margin.
